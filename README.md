## Как собрать и запустить

У меня установлены `Windows 10` и `g++ (MinGW.org GCC-6.3.0-1) 6.3.0`. \
Ниже описаны шаги по установке и запуску.

```batch
g++ main.cpp -fopenmp -o main.exe
main.exe
```
